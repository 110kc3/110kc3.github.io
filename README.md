# 110kc3.github.io

Domain-root discovery surface for this GitHub account's project sites — created because AI crawlers and agent-readiness auditors look for `llms.txt`, `robots.txt`, `sitemap.xml`, `agents.json` and `.well-known/*` at the **domain root**, which a project site (like `/seo/`) can't serve.

Everything here points at the main resource: **[AI Product Index](https://110kc3.github.io/seo/)** (repo: [110kc3/seo](https://github.com/110kc3/seo)).

Static files only, no build. Safe to extend into a fuller user site later — just keep the discovery files.
